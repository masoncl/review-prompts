# CM-002: Data format and return value changes

**Risk**: Logic error

**When to check:** Mandatory when data encoding or interpretation changes for a variable or structure

Place each step defined below into TodoWrite.

**Mandatory data format validation:**
- step 1: read entire function
  - Add function name to TodoWrite
  - Output: function name and location
- step 2: check all data encoding in modified functions, even if they didn't change
  - Add data encoding description to TodoWrite
  - Output encoding line
- step 3: when changing how data is encoded/interpreted within a function, verify consistency
- step 4: read entire function of any callers/callees if new data encoding is returned or passed down
- step 5: verify new encodings used correctly in new location
